# AstraWare Client — Static Site

Plain HTML/CSS/JS export of the AstraWare landing page. No build step required.

## Files
- `index.html` — markup + TikTok embed script
- `styles.css` — all styling (dark theme, white accents)
- `astraware-logo.png` — logo asset

## Run locally
Just open `index.html` in a browser, or serve the folder:
```bash
python3 -m http.server 8000
```
Then visit http://localhost:8000

## Deploy
Upload the folder to any static host: GitHub Pages, Netlify, Vercel, Cloudflare Pages, etc.
